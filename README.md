# jira-mini
A Jira-inspired task and project management system for teams. Built as a full-stack portfolio project using Spring Boot and React.

## Goals
This project aims to simulate a real-world Agile project management tool.

Main objectives:
- Practice designing a multi-role system (RBAC)
- Build a full-stack application from scratch
- Apply real development workflow (docs → design → API → code)
- Prepare for graduation thesis and job portfolio


## Core Features
### Workspace Management
- Create and manage workspaces
- Invite members
- Role-based access control

### Project Management
- Multiple projects per workspace
- Project member management

### Task Management
- Kanban board
- Create / assign / update tasks


## Tech Stack
### Backend
- Spring Boot
- Spring Security (JWT)
- MySQL
- JPA / Hibernate

### Frontend (planned)
- React
- Javascript

### Role System
- Admin
- Workspace Owner
- Project Manager
- Member
